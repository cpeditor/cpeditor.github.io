Редактор, которого не хватало
спортивным программистам

CP Editor специально разработан для спортивного программирования. Он помогает вам сконцентрироваться на алгоритмах и автоматизирует компиляцию, выполнение и тестирование кода, а также умеет отображать тесты с различных платформ и может отправлять решения на Codeforces!

Смотреть на GitHub Скачать

Как это работает?

CP Editor избавляет вас от написания рутинного кода пока идёт соревнование. Он автоматизирует большинство процессов, позволяя сосредоточиться на решении задачи.

Захват тестов

CP Editor способен получать примеры автоматически почти с любого сайта соревнований. Можете попрощаться с ошибками при копировании тестов.

Быстрая печать

Пишите быстрее, используя крутые возможности шаблонов. Буквально за две секунды вы можете вставить целый DFS в свой код.

Получение вердикта

Вы можете получить ответ на ваши примеры в один клик при помощи встроенной проверки или же из testlib. Если они вам не подходят, то добавьте свои варианты.

  • Запуск без сохранения
  • Множество тем
  • Проверка кода
  • Отправка на Codeforces
